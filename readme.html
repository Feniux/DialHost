Briefing do teste para a DialHost


Criar uma landing page com captura de dados do visitante e gravá-lo no mySql.
Exemplo: Nome, Email, Celular, Nascimento e Endereço.
Não é necessário fazer uma listagem dos usuários cadastrados.
Segue as diretrizes:

  1. Utilizar Laravel (Versão mais recente)
  2. Montar um Migration para iniciar o banco de dados
  3. Desenvolver uma landing page responsiva contendo o formulário
  4. Todos os campos devem ser validados dentro de valores reais e possiveis
  5. Os dados de endereço devem ser preenchidos consumindo a API : API Correios
  6. Enviar o link do repositório com o resultado até 04/06/2017 (Domingo)
